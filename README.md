## 🚀 Getting Started

To run this portfolio locally:

```bash
# Clone the repository
git clone https://github.com/manjunathshiva/portfolio.git

# Navigate to project directory
cd portfolio

# Install dependencies
npm install

# Start development server
npm start
```
